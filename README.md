# Unity Font Replacer — Thai Edition

A fork of [snowyegret23/Unity_Font_Replacer](https://github.com/snowyegret23/Unity_Font_Replacer) adapted for **Thai language font replacement** in Unity games.

Replace fonts in compiled Unity game asset bundles with Thai fonts (Sarabun, Noto Sans Thai, or any custom Thai TTF) — no source code required.

---

## Credits

Original work by **snowyegret23**
→ https://github.com/snowyegret23/Unity_Font_Replacer

This fork adds Thai language support:
- `--sarabun` / `--notosansthai` bulk replacement modes
- `CharList_Thai.txt` — Thai Unicode character set (U+0E00–U+0E7F + ASCII)
- `TH_ASSETS/` — folder for placing Thai font files
- Thai-first asset resolution (`TH_ASSETS/` is checked before `KR_ASSETS/`)
- Updated `make_sdf.py` default charset to `CharList_Thai.txt`

## Requirements

- Python 3.12+
- `pip install UnityPy TypeTreeGeneratorAPI Pillow scipy`

## Thai Font Setup

Place your Thai font files inside the `TH_ASSETS/` folder:

```
TH_ASSETS/
  Sarabun.ttf
  NotoSansThai.ttf
  Sarabun SDF.json          ← generated by make_sdf.py
  Sarabun SDF Atlas.png     ← generated by make_sdf.py
  Sarabun SDF Material.json ← generated by make_sdf.py
```

Recommended free Thai fonts:
- [Sarabun](https://fonts.google.com/specimen/Sarabun) — Google Fonts
- [Noto Sans Thai](https://fonts.google.com/noto/specimen/Noto+Sans+Thai) — Google Fonts

## Usage

### 1. Generate SDF atlas from TTF (for TextMeshPro games)

```bash
python make_sdf.py --ttf TH_ASSETS/Sarabun.ttf
```

Output files (`Sarabun SDF.json`, `Sarabun SDF Atlas.png`, `Sarabun SDF Material.json`) will be placed in the current directory — move them into `TH_ASSETS/`.

### 2. Scan game fonts (create JSON map)

```bash
python unity_font_replacer_th.py --gamepath "C:/path/to/game" --parse
```

### 3. Bulk replace with Sarabun

```bash
python unity_font_replacer_th.py --gamepath "C:/path/to/game" --sarabun
```

### 4. Bulk replace with Noto Sans Thai

```bash
python unity_font_replacer_th.py --gamepath "C:/path/to/game" --notosansthai
```

### 5. Replace specific fonts using JSON map

Edit the generated JSON to map each original font to a replacement font, then:

```bash
python unity_font_replacer_th.py --gamepath "C:/path/to/game" --list font_map.json
```

### Options

| Flag | Description |
|---|---|
| `--sarabun` | Bulk replace all fonts with Sarabun |
| `--notosansthai` | Bulk replace all fonts with Noto Sans Thai |
| `--sdfonly` | Replace SDF (TextMeshPro) fonts only |
| `--ttfonly` | Replace TTF fonts only |
| `--parse` | Export font info to JSON |
| `--list <file>` | Replace fonts using a JSON mapping file |
| `--ps5-swizzle` | Enable PS5 texture swizzle/unswizzle mode |
| `--preview-export` | Export atlas preview PNGs before replacing |
| `--scan-jobs <n>` | Number of parallel scan workers |

## Extract existing fonts

```bash
python export_fonts_th.py --gamepath "C:/path/to/game"
```

Exports all TMP SDF font assets (JSON + PNG atlas) from the game.

---

## Interactive mode

Run without flags to use the interactive menu:

```bash
python unity_font_replacer_th.py --gamepath "C:/path/to/game"
```

```
Select a task:
  1. Export font info (create JSON)
  2. Replace fonts using JSON
  3. Bulk replace with Sarabun (Thai)
  4. Bulk replace with Noto Sans Thai
  5. Bulk replace with Mulmaru
  6. Bulk replace with NanumGothic
  7. Preview export (Atlas/Glyph crops)
```
